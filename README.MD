# Code-Youdle!
This serves as a real-world example of building a production application in a team setting. Mentors will lead students through defining requirements, whiteboarding, sprint planning, and developing a Wordle clone. This exercise is being done from a "start up" perspective and as part of Code-You's Web Development course.

## Contribution Guidelines
Let's not get too complicated. This is an educational project after all. 
- Create fork
- update `main` branch on your fork
- Create pull request to Code You's organization repo
- Rinse and repeat!

## Educational Notes
### Why Wordle?
Wordle, at it's core, is a very simple application. Since it already exists, it allows the "Team Leads" (Mentors) to run the demonstrations in an accelerated timeline. There's no need to spend weeks-months on ideation and defining an MVP like real startups might. It also allows students to compare their work to the existing product.

### Disclaimer
This is meant to be a clone of the New York Time's Wordle game. This is for educational purposes only. There is no intention to compete or use the application as a source of monetary income.
